# Videira
<br>
Using OOP access a dict values as attributes.

<br>
<br>

## Usage example

```python

    from videira import Cacho

    foo = { 'a':[1,2,3], 'b': {'c': 2} }
    
    bar = Cacho(foo)
    
    print(bar)
    print(bar.a)
    print(bar.a.b.c)
```
